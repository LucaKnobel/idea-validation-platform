# Evidara

## Overview

Evidara is a web application for the structured validation of digital business ideas.

It helps users:

- define ideas
- create hypotheses
- run experiments
- collect evidence
- evaluate validation results

---

## Status

Work in Progress

- under active development
- features and APIs may change
- not production-ready

---

## Tech Stack

- Nuxt 4
- TypeScript
- Prisma + PostgreSQL
- Tailwind CSS
